## ðŸŽ¯ Common Tasks

### Add a New Tour Package
1. Click **Services > Tours & Packages**
2. Click "Add New Tour"
3. Fill in:
   - Tour name and description
   - Duration and pricing
   - Itinerary details
   - Images
4. Set availability dates
5. Click "Publish"

### Manage Bookings
1. Go to **Management > Bookings**
2. View all bookings in the list
3. Filter by:
   - Date range
   - Status (pending, confirmed, completed)
   - Tour type
4. Click on a booking to:
   - View details
   - Update status
   - Send confirmation email
   - Add notes

### Edit Homepage Hero
1. Navigate to **Landing Page CMS > Hero Section**
2. Update:
   - Main headline
   - Subheadline
   - Call-to-action button text/link
   - Background image/video
3. Preview changes
4. Click "Save"

### Add a Blog Post
1. Go to **Blog System > Blog Manager**
2. Click "Create New Post"
3. Enter:
   - Title and slug
   - Content (use rich text editor)
   - Featured image
   - Categories and tags
   - SEO metadata
4. Preview
5. Click "Publish" or "Save Draft"

### Use AI Content Generator
1. Navigate to **Blog System > AI Generator**
2. Enter:
   - Topic or keywords
   - Content type (blog post, tour description, etc.)
   - Tone (professional, friendly, exciting)
3. Click "Generate"
4. Review and edit generated content
5. Save to blog or content library
